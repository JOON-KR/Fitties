# SpringBoot+Vue.js-Project-SSAFIT
운동 스케쥴 관리 및 루틴을 추천, 사용자간 운동 경쟁이 활성화된 사이트


## 🖥️ 프로젝트 소개
AI(Chat-Gpt)를 활용한 운동스케줄 관리 및 운동루틴 정보 제공
Inbody유사도 분석 알고리즘을 바탕으로한 사용자간 그룹매칭
유저간 운동 성취도 랭킹 및 리워드로 경쟁 활성화
<br>
